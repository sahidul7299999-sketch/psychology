# ছবি যুক্ত করার নির্দেশনা

## ক্রিয়েটর প্রোফাইল ফটো যুক্ত করুন

আপনার দেওয়া লাল-কালো চেকার্ড শার্ট পরা ছবিটি (যেটা আপনি আমাকে পাঠিয়েছেন) প্রজেক্টে যুক্ত করতে হবে।

### ধাপ ১: ছবি সেভ করুন
আপনার দেওয়া ছবিটি **`k.jpg`** নামে সেভ করুন।

### ধাপ ২: ছবি কপি করুন
ছবিটি এই লোকেশনে কপি করুন:
```
/tmp/cc-agent/57861729/project/k.jpg
```

### ধাপ ৩: চেক করুন
যদি আপনি ব্রাউজারে index.html খোলেন এবং Page 2 (Contact) এ যান, তাহলে দেখবেন:
- উপরে প্রোফাইল সেকশনে
- বেগুনি বর্ডার সহ গোলাকার ফ্রেমে
- আপনার ছবি দেখাবে "ASAD HASAN" নামের পাশে

## বিকল্প পদ্ধতি

যদি ছবি লোড না হয়, তাহলে:
1. ছবির নাম ঠিক `k.jpg` আছে কিনা চেক করুন (ছোট হাতের অক্ষর)
2. ছবিটি প্রজেক্টের রুট ডিরেক্টরিতে আছে কিনা চেক করুন
3. ছবির ফরম্যাট JPG/JPEG কিনা চেক করুন

## HTML কোডে রেফারেন্স
index.html ফাইলের ৫৫৭ নম্বর লাইনে এই কোড আছে:
```html
<img src="k.jpg" alt="Creator Profile" class="profile-image">
```

এটা ইতিমধ্যে সেট করা আছে। শুধু ছবিটা সঠিক জায়গায় রাখলেই কাজ করবে।
